# <a name="fields-overview"></a>필드 개요

필드는 엔터티에 정보를 저장하는 데 사용할 수 있는 개별 데이터 항목을 정의합니다. 기존 표준 엔터티에 요구 사항을 충족하는 필드가 없으면 데이터를 캡처할 새 필드를 만듭니다. 새 필드를 만든 후에는 엔터티의 해당 양식 및 보기에 포함시켜 앱에서 사용할 수 있도록 해야 합니다.
