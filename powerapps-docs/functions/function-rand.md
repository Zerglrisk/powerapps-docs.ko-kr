# <a name="rand-function-in-powerapps"></a>PowerApps의 Rand 함수
의사 난수를 반환합니다.

## <a name="description"></a>설명
**Rand** 함수는 0보다 크거나 같지만 1보다 작은 의사 난수를 반환합니다.

## <a name="syntax"></a>구문
**Rand**()
