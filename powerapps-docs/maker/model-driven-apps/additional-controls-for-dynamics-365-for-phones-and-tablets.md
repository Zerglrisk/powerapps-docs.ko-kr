# <a name="additional-controls-for-dynamics-365-for-phones-and-tablets"></a>Dynamics 365 for phones 및 tablets 추가 컨트롤 

 다양한 추가 컨트롤을 사용하여 Dynamics 365 for phones 및 tablets에서 보다 터치가 용이한 환경을 만들 수 있습니다. 여기에는 슬라이더, 스위치, 멀티미디어 플레이어, 입력 마스크, 일정 및 기타 컨트롤이 포함됩니다.  

 
> [!NOTE]
>  모바일 앱에만 이러한 추가 컨트롤을 사용할 수 있습니다. 웹 응용 프로그램에서는 지원되지 않습니다.  
  
 양식 편집기에서 이러한 컨트롤을 사용하려면:  
  
1.  컨트롤을 추가할 필드 또는 명령을 두 번 클릭합니다.  
  
2.  **컨트롤** 탭을 클릭합니다.  
  
3.  **컨트롤 추가**를 클릭합니다.  
  
4.  원하는 컨트롤을 선택한 다음 **추가**를 클릭합니다.  
  
    > [!NOTE]
    >  필드 또는 목록 유형에 따라 다른 컨트롤을 사용할 수 있습니다. 예를 들어, 슬라이더 컨트롤은 숫자 또는 비용 필드에만 사용할 수 있고 일정 컨트롤은 목록에만 사용할 수 있습니다.  
  
5.  (휴대폰, 태블릿, 또는 둘 다)에 컨트롤을 표시하려는 장치를 선택합니다. 컨트롤은 전화 헤더 필드에 사용할 수 없습니다.  
  
6.  각 속성의 값을 구성합니다.  
  
7.  컨트롤 구성이 완료되면 **확인**을 클릭합니다.  
  
 다음은 Dynamics 365 for phones 및 tablets에서 양식에 사용할 수 있는 각 컨트롤에 대한 설명입니다.  
  
## <a name="calendar-control"></a>일정 컨트롤  
 이 컨트롤을 사용하여 Dynamics 365 for phones 및 tablets에서 일정 보기로 나타나도록 양식을 구성합니다. 휴대폰 및 태블릿용 대시보드, 목록 또는 엔터티 표를 대체하기 위해 이 컨트롤을 사용할 수도 있습니다.  
  
|속성|설명|  
|--------------|-----------------|  
|시작 날짜|일정 보기에서 시각화할 항목의 시작 날짜 및 시간을 정의합니다. 사용 가능한 값은 이런 유형의 날짜 보기에 있는 열입니다.|  
|종료 날짜|일정 보기에서 시각화할 항목의 종료 날짜 및 시간을 정의합니다. 사용 가능한 값은 이런 유형의 날짜 보기에 있는 열입니다.|  
|기간|기간(분). 종료 날짜에 대한 값을 지정하는 경우 기간은 무시됩니다.|  
|설명|이는 일정 항목에 대해 보려는 캡션입니다.|  
  
 일정에 표시되는 최소 기간은 30분입니다. 기간이 30분 미만인 항목은 30분으로 나타납니다.  
  
 일정 컨트롤은 모든 날짜 동작(사용자 현지 시간, 날짜만 및 표준 시간대 독립)을 지원합니다.

|CC_Timeline_Title_Desc|제목 설명:|

|CC_Timeline_Label1_Desc|레이블 1 설명.|  
|CC_Timeline_Label2|Label 1 다음에 표시될 필드.|  
|CC_Timeline_Label2_Desc|레이블 2 설명.|  
|CC_Timeline_Label3|Label 2 다음에 표시될 필드.|  
|CC_Timeline_Label3_Desc|레이블 3 설명.|  
|CC_Timeline_Label4|Label 3 다음에 표시될 필드.|  
|CC_Timeline_Label4_Desc|레이블 4 설명.|  
|CC_Timeline_Label5|Label 4 다음에 표시될 필드.|  
|CC_Timeline_Label5_Desc|레이블 5 설명.|  
|CC_Timeline_Timestamp|역순으로 일정 분류에 사용할 필드.|  
|CC_Timeline_Timestamp_Desc|타임 스탬프 설명.|  
|CC_Timeline_Group|그룹핑 일정에 대해 매핑할 필드.|  
|CC_Timeline_Group_Desc|그룹 필드 설명.|  
|CC_Timeline_GroupOrder|관련된 다른 그룹에 속한 항목 그룹 순서(표시되는 그룹에 대해 1, 2, 3 등의 값 할당). 할당된 그룹 값의 오름 차순으로 그룹이 표시됩니다.|  
|CC_Timeline_GroupOrder_Desc|그룹 순서 필드 설명.|  
|CC_Timeline_URL|각 일정 항목의 URL 표시에 대해 매핑할 URL 필드.|  
|CC_Timeline_URL_Desc|URL 필드 설명.|  
|CC_Timeline_ThumbnailURL|각 항목에 표시할 이미지/아이콘의 축소판에 대해 매핑할 필드.|  
|CC_Timeline_ThumnailURL_Desc|`ThumbnailURL` 필드 설명.|  
|CC_Timeline_Filter|일정 필터에 대해 매핑할 필드.|  
|CC_Timeline_Filter_Desc|필터 설명.|  
|CC_Timeline_Footer|일정 바닥글로 표시할 웹 리소스.|  
|CC_Timeline_Footer_Desc|바닥글 필드 설명.|  
  
## <a name="linear-slider"></a>선형 슬라이더  
 선형 슬라이더 컨트롤을 사용하면 슬라이더를 끌어 숫자 값을 입력할 수 있으며 수량을 입력하는 옵션도 제공합니다. 슬라이더는 정수 입력 및 표시만 제공합니다. 숫자 또는 금액 필드에 대해 이 컨트롤을 사용합니다.  
  
|속성|설명|  
|--------------|-----------------|  
|최대|슬라이더에 표시할 최대 값을 설정합니다.|  
|분|슬라이더에 표시할 최소 값을 설정합니다.|  
|값|슬라이더에 표시할 값을 설정합니다.|  
|단계|이 컨트롤을 사용하여 데이터를 입력할 때 현재 값에서 더하거나 뺄 양을 설정합니다.|  
  
## <a name="option-sets"></a>옵션 집합  
 옵션 집합 컨트롤은 사용자가 데이터를 입력할 때 선택할 수 있는 선택 집합을 제공합니다. 두 개 또는 세 개 옵션이 있는 옵션 집합에 대해 이 컨트롤을 사용합니다.  
  
|속성|설명|  
|--------------|-----------------|  
|필드|컨트롤이 매핑되는 필드를 표시합니다.|  
  
## <a name="flip-switch"></a>플립 전환  
 플립 스위치는 켜기/끄기 스위치와 같으며 두 값 중에서 선택할 수 있습니다.  
  
|속성|설명|  
|--------------|-----------------|  
|필드|컨트롤이 매핑되는 필드를 표시합니다.|  
  
## <a name="star-rating"></a>별 등급  
 이 별 등급을 사용하여 등급의 시각적 표현을 제공합니다. 설정할 수 있는 별의 최대 개수는 5입니다. 정수의 경우에만 이 컨트롤을 사용할 수 있으며 10진수 값은 사용할 수 없습니다.  
  
> [!NOTE]
>  이 컨트롤에 대해 **웹에서 숨기기** 옵션을 선택합니다.  
  
|속성|설명|  
|--------------|-----------------|  
|최대|드롭다운 목록에서 컨트롤의 최대 별 개수를 선택합니다.|  
  
## <a name="radial-knob"></a>방사형 노브  
 방사형 노브는 사용자가 노브를 움직여 데이터를 입력하는 방법을 제공하며 화면에 원형으로 나타납니다. 방사형 노브 컨트롤은 정수 입력 및 표시만 제공합니다. 숫자 또는 금액 필드에 대해 이 컨트롤을 사용합니다. 터치를 사용하여 값을 변경하거나 키패드를 사용하여 숫자 값에 포커스를 맞추고 편집할 수 있습니다.  
  
> [!NOTE]
>  이 컨트롤은 Android 4.2 및 4.3 기기에서 지원되지 않습니다. 해당 버전에서 이러한 컨트롤은 스크롤 환경에 영향을 미칩니다.  
  
|속성|설명|  
|--------------|-----------------|  
|최대|게이지에 표시할 최대 값을 설정합니다.|  
|분|게이지에 표시할 최소 값을 설정합니다.|  
|값|게이지에 표시할 값을 가져오거나 설정합니다.|  
|단계|이 컨트롤을 사용하여 데이터를 입력할 때 현재 값에서 더하거나 뺄 양을 설정합니다.|

>  [개인정보취급방침](use-the-form-editor-legacy.md#BKMK_PrivacyNotices)  
  
|속성|설명|  
|--------------|-----------------|  
|필드|컨트롤이 매핑되는 필드를 표시합니다.|  
  
## <a name="bullet-graph"></a>불릿 그래프  
 불릿 그래프 컨트롤은 측정이 양호, 불량 또는 다른 상태인지 여부를 신호하기 위해 비교 측정과 정성적 범위를 사용한 단일 키 측정을 표시합니다. 대시보드에서 숫자 또는 금액 필드에 대해 이 컨트롤을 사용합니다. 예를 들어 값을 실제 수익에 매핑하고 대상을 예상 수익에 매핑하여 실제 수익과 예상 수익을 비교하여 시각화할 수 있습니다.  
  
|속성|설명|  
|--------------|-----------------|  
|최대|그래프에 표시할 최대 값을 설정합니다.|  
|분|그래프에 표시할 최소 값을 설정합니다.|  
|적절함|측정 단위에 적절하다고 생각하는 값을 설정합니다(선택).|  
|적절하지 않음|측정 단위에 부적절하다고 생각하는 값을 설정합니다(선택).|  
|값|컨트롤이 매핑되는 필드를 표시합니다.|  
|대상|값을 비교할 필드에 매핑합니다. 예를 들어 **값**이 **실제 수익**에 매핑된 경우 **대상**을 **예상 수익**에 매핑할 수 있습니다. 그렇지 않으면 정적 값을 제공합니다.|  
  
## <a name="pen-control"></a>펜 컨트롤  
 서명과 같은 필기 입력을 캡처하는 펜 컨트롤을 사용합니다.  
  
> [!NOTE]
>  이 컨트롤을 매핑하는 필드에 지정된 권장하는 최소 **최대 길이**는 15000입니다.  
>   
>  이 컨트롤에 대해 **웹에서 숨기기** 옵션을 선택합니다.  
  
|속성|설명|  
|--------------|-----------------|  
|PenMode|**PenMode!Draw**, **PenMode!Erase** 또는 **PenMode!Select**를 지정하여 사용자가 펜 컨트롤에서 포인팅 장치를 끌 때 어떤 일이 발생하는지 확인합니다.|  
  
## <a name="auto-complete"></a>자동 완성  
 자동 완성 컨트롤은 입력할 때 항목 목록을 필터링하고 드롭다운 목록에서 값을 선택할 수 있습니다. 예를 들어, 이 컨트롤을 사용하여 주 또는 국가/지역의 드롭다운 목록에서 선택할 수 있습니다. 이 컨트롤은 **한 줄 텍스트** 유형 필드에 매핑됩니다.  
  
|속성|설명|  
|--------------|-----------------|  
|필드|컨트롤이 매핑되는 필드를 표시합니다.|  
|출처|데이터 소스를 설정합니다(그룹 옵션, 옵션 집합 또는 보기).|  
|옵션 집합|이 필드에 대한 옵션 집합을 선택합니다.|  
|보기|이 필드에 대한 엔터티와 보기를 선택합니다.|  
|필드|데이터 원본으로 사용할 보기의 기본 엔터티의 필드를 선택합니다.|  
  
## <a name="multimedia"></a>멀티미디어  
 이동 중 판매 및 현장 직원들을 위한 풍부한 고객 경험을 제공하기 위한 비디오를 포함할 수 있습니다. 이 컨트롤을 사용하여 컨트롤에서 재생할 오디오 또는 비디오 링크를 포함하는 URL 필드에 매핑합니다.  
  
> [!NOTE]
>  이 컨트롤은 Android 버전 4.4 이상에서 지원됩니다.  
>   
>  YouTube 동영상은 현재 Windows 8 및 Windows 8.1 태블릿 및 휴대폰에서 지원되지 않습니다. Windows 10에서는 YouTube를 포함하여 HTTPS 동영상만 현재 지원됩니다.  
  
 지원되는 미디어 유형은 다음과 같습니다.  
  
-   스트리밍 MP4 파일  
  
-   YouTube 비디오  
  
-   Azure 미디어  
  
-   오디오 스트림  
  
 [개인정보취급방침](use-the-form-editor-legacy.md#BKMK_PrivacyNotices)  
  
|속성|설명|  
|--------------|-----------------|  
|미디어|이 컨트롤에서 재생할 미디어의 URL을 입력합니다.|  
  
## <a name="number-input"></a>숫자 입력  
 숫자 입력 컨트롤을 사용하여 사용자가 데이터를 신속하게 입력할 수 있도록 합니다. 사용자가 설정한 증분으로 숫자 값을 변경하려면 더하기와 빼기 단추만 눌러야 합니다. 숫자 또는 금액 필드에 대해 이 컨트롤을 사용합니다. 또한 사용자가 필드에 직접 숫자를 입력할 수도 있습니다. 이 필드는 편집 모드에서에서만 지원됩니다.  
  
|속성|설명|  
|--------------|-----------------|  
|단계|이 컨트롤을 사용하여 데이터를 입력할 때 현재 값에서 더하거나 뺄 양을 설정합니다.|  
|필드|컨트롤이 매핑되는 필드를 표시합니다.|  
  
## <a name="input-mask"></a>입력 마스크  
 입력 마스크 컨트롤을 사용하여 전화 번호 또는 신용 카드 같은 필드에 대해 서식을 설정하여 잘못된 데이터 입력을 방지합니다. 예를 들어, 형식 +1-222-555-1011로 미국 전화 번호를 입력하도록 하려면 입력 마스크 +1-000-000-0000를 사용합니다.  
  
|속성|설명|  
|--------------|-----------------|  
|마스크|사용자가 입력할 때 데이터의 유효성을 검사할 마스크를 입력합니다. 마스크에 대해 다음 문자 조합을 사용할 수 있습니다.<br /><br /> 0 - 숫자<br /><br /> 9 - 숫자 또는 공백<br /><br /> # – 숫자, 부호 또는 공백<br /><br /> L – 문자<br /><br /> I – 문자 또는 공백<br /><br /> A – 영숫자<br /><br /> A – 영숫자 또는 공백<br /><br /> < – 소문자로 문자 변환<br /><br /> > – 대문자로 문자 변환<br /><br /> &#124; – 대/소문자 변환 사용 안 함<br /><br /> \ – 모든 문자를 리터럴로 변환하여 벗어남<br /><br /> 기타 모두 - 리터럴|  
|필드|컨트롤이 매핑되는 필드를 표시합니다.|  
  
## <a name="linear-gauge"></a>선형 게이지  
 선형 게이지를 사용하면 정확한 수량을 입력하는 대신 슬라이더를 끌어 숫자 값을 입력할 수 있습니다. 슬라이더는 정수 입력 및 표시만 제공합니다. 숫자 및 금액 필드에 대해 이 컨트롤을 사용합니다.  
  
|속성|설명|  
|--------------|-----------------|  
|최대|게이지에 표시할 최대 값을 설정합니다.|  
|분|게이지에 표시할 최소 값을 설정합니다.|  
|값|게이지에 표시할 값을 가져오거나 설정합니다.|  
|단계|이 컨트롤을 사용하여 데이터를 입력할 때 현재 값에서 더하거나 뺄 양을 설정합니다.|  
  
## <a name="arc-knob"></a>호 노브  
 호 노브는 사용자가 노브를 움직여 데이터를 입력하는 방법을 제공하며 화면에 호로 나타납니다. 호 노브 컨트롤은 정수 입력 및 표시만 제공합니다. 숫자 및 금액 필드에 대해 이 컨트롤을 사용합니다. 터치를 사용하여 값을 변경할 수 있으며 키패드를 사용하여 숫자 값에 포커스를 맞추고 편집할 수 있습니다.  
  
> [!NOTE]
> 이 컨트롤은 Android 4.2 및 4.3 기기에서 지원되지 않습니다. 해당 버전에서 이러한 컨트롤은 스크롤 환경에 영향을 미칩니다.  
  
|속성|설명|  
|--------------|-----------------|  
|최대|게이지에 표시할 최대 값을 설정합니다.|  
|분|게이지에 표시할 최소 값을 설정합니다.|  
|값|게이지에 표시할 값을 가져오거나 설정합니다.|  
|단계|이 컨트롤을 사용하여 데이터를 입력할 때 현재 값에서 더하거나 뺄 양을 설정합니다.|
