# <a name="entity-overview"></a>엔터티 개요

엔터티는 비즈니스 데이터를 모델링하고 관리하는 데 사용됩니다. 앱을 개발할 때 표준 엔터티, 사용자 지정 엔터티 또는 둘 다 사용할 수 있습니다. 앱용 CDS는 기본적으로 표준 엔터티를 제공합니다. 이는 모범 사례에 따라 조직 내에서 가장 일반적인 개념과 시나리오를 캡처하기 위해 설계되었습니다.
