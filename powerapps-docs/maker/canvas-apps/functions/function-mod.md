# <a name="mod-function-in-powerapps"></a>PowerApps의 Mod 함수
나눗셈의 나머지를 반환합니다.

## <a name="description"></a>설명
**Mod** 함수는 숫자를 제수로 나눈 나머지를 반환합니다.

결과는 제수와 같은 부호를 갖습니다.

## <a name="syntax"></a>구문
**Mod**( *Number*, *Divisor* )

* *Number* - 필수 항목입니다. 나눌 숫자입니다.
* *Divisor* - 필수 항목입니다.  제수입니다.
