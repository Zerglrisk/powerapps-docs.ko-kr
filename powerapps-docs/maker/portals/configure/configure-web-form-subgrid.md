# <a name="configure-web-form-subgrids-for-portals"></a>포털의 웹 양식 하위 표 구성

웹 양식 하위 표는 엔터티 양식 하위 표와 동일한 방식으로 구성됩니다. 먼저, 하위 표가 있는 웹 양식 단계를 위한 메타데이터 레코드를 만들고 구성 메타데이터를 추가합니다.

포털에서 관리 양식에 하위 표를 추가하는 방법은 간단합니다. 기본 제공 양식 디자이너를 사용하여 관리하는 양식에 하위 표를 추가하기만 하면 됩니다. 표에서는 Common Data Service 양식 디자이너에 지정된 보기를 사용하고, 해당 옵션이 선택된 경우 관련 레코드만 표시하고, 선택적으로 검색 막대를 표시하고, [포털에 대한 엔터티 권한](assign-entity-permissions.md)을 준수합니다. 레코드의 읽기 전용 목록을 표시하는 것은 더욱 간단합니다. 표에 관한 작업(예: 만들기, 업데이트, 삭제 등)을 사용하려면, 메타데이터 구성을 사용하여 이러한 작업을 구성해야 합니다.

## <a name="add-subgrid-metadata-to-your-form"></a>양식에 하위 표 메타데이터 추가

엔터티 양식에 하위 표 메타데이터를 추가하기 위해 상단 드롭다운 목록 또는 작업하고 있는 레코드의 기본 양식에 있는 하위 표를 사용하여 **엔터티 양식 메타데이터**로 이동합니다. 추가 정보: [엔터티 양식 정의](entity-forms.md).

새 레코드를 추가하려면 **새 엔터티 양식 메타데이터 추가**를 선택합니다.

기존 레코드를 편집하려면 표에서 레코드를 선택합니다. **유형** 값으로 **하위 표**를 선택하면 다른 특성인 **하위 표 이름**이 표시됩니다.


|     이름     |                                                       설명                                                        |
|--------------|--------------------------------------------------------------------------------------------------------------------------|
| 하위 표 이름 | 엔터티의 관련된 양식에서 하위 표의 고유 이름입니다. |
|              |                                                                                                                          |

양식 편집기에서 하위 표를 선택하면 속성 창이 표시됩니다. 여기에는 엔터티 양식 메타데이터 레코드의 **하위 표 이름** 필드에 할당하는 데 사용할 **이름** 필드가 포함되어 있습니다.

![하위 표 메타데이터 추가](../media/add-subgrid-metadata.png "하위 표 메타데이터 추가")  

올바른 하위 표 이름을 지정하면 하위 표 구성 설정이 표시됩니다. 기본적으로 **기본 설정**만 표시됩니다. **고급 설정**을 선택하여 추가 설정을 표시합니다.

기본적으로 공간을 절약하기 위해 대부분의 설정은 축소 표시됩니다. 섹션을 확장하고 추가 옵션을 보려면 **** 를 클릭하십시오. **** 를 선택하여 섹션을 축소합니다.

## <a name="create-action"></a>동작 만들기

**동작 만들기**를 사용하면 선택했을 때 사용자가 새 레코드를 만들 수 있는 [엔터티 양식](entity-forms.md)과 함께 대화 상자를 표시하는 단추가 하위 표 위쪽에 렌더링됩니다.  

### <a name="create-action-settings"></a>만들기 동작 설정

| 이름                  | 설명                                                                                                                                                                                                                                                 |
|-----------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **기본 설정**    |                                                                                                                                                                                                                                                             |
| 엔터티 양식           | 새 레코드를 만드는 데 사용할 [엔터티 양식 및 사용자 지정 논리](entity-forms.md)를 지정합니다. 드롭다운 목록에는 하위 표의 엔터티 유형에 대해 구성된 모든 엔터티 양식이 나열됩니다.<br>**참고**: 하위 표의 엔터티 유형에 엔터티 양식이 없는 경우 빈 드롭다운 목록이 나타납니다. 만들기 동작에 엔터티 양식이 제공되지 않은 경우에는 무시되며 하위 표의 엔터티 양식에 단추가 렌더링되지 않습니다.                                |
| **고급 설정** |                                                                                                                                                                                                                                                             |
| 단추 레이블          | 하위 표 위쪽의 만들기 동작 단추에 표시되는 HTML 레이블을 다시 정의합니다.                                                                                                                                                                           |
| 단추 도구 모음        | 사용자가 만들기 동작 단추를 가리킬 때 나타나는 도구 설명 텍스트를 다시 정의합니다.                                                                                                                                                            |

| 해제 단추 화면 읽기 프로그램 텍스트 | 대화 상자의 해제 단추와 관련된 화면 뷰어 텍스트를 다시 정의합니다.                                                                   |
