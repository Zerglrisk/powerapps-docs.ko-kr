# <a name="fields-overview"></a>필드 개요

필드는 엔터티에 정보를 저장하는 데 사용할 수 있는 개별 데이터 항목을 정의합니다. 기존 표준 엔터티에 사용자 요구 사항에 맞는 필드가 없을 경우 새 필드를 만들어 데이터를 캡처합니다. 새 필드를 만든 다음에는 앱에서 사용할 수 있도록 적절한 엔터티의 양식 및 보기에 필드를 포함해야 합니다.
